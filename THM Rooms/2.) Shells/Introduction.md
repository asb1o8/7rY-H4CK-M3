## Overview

- A shell is software that allows a user to interact with an OS. It can be a graphical interface, but it is usually
  a CLI, and this will depend on the operating system running on the target system.
  Some usecases of Shell an attacker use:-
  - Remote System Control
  - Privilege Escalation
  - Data Exfiltration
  - Persistence & Maintenance Access
  - Post-Exploitation Activities
  - Access other Systems on the network
